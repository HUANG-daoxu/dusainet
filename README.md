# 简介
这是一个由杜赛开发的博客网站Django项目。

网站链接在这里：[杜赛的个人网站](http://www.dusaiphoto.com)

里面包含博客、教程、读书、照片墙几个板块功能。

作者是一个光学工程师，编程是业余爱好。这也是我写的第一个Django项目，欢迎各位前辈提出宝贵的意见。

作者邮箱：dusaiphoto@foxmail.com
